🍎 Food Spoilage Prediction System
📌 Overview

The Food Spoilage Prediction System is a machine learning-based application designed to determine whether a food item is fresh or spoiled based on input parameters. The system integrates a Flask backend with an intuitive Gradio interface, enabling real-time predictions through an interactive web-based UI.

🚀 Key Features
Predicts food spoilage using a trained machine learning model
Interactive and user-friendly interface powered by Gradio
Lightweight and efficient backend using Flask
Real-time prediction capability
Modular and scalable project structure
🧠 Technology Stack
Programming Language: Python
Backend Framework: Flask
Frontend Interface: Gradio
Machine Learning: Scikit-learn
Data Processing: Pandas, NumPy

⚙️ Installation and Setup
1. Clone the Repository
git clone https://github.com/Suraj03-ms/food-spoilage-prediction.git
cd food-spoilage-prediction
2. Create and Activate Virtual Environment
python -m venv venv
venv\Scripts\activate
3. Install Dependencies
pip install -r requirements.txt
4. Run the Application
python app.py
